ASIMS

Afghanistan Statistical Information Management System

What is ASIMS

  • ASIMS is an information management system for surveys.
  • It receives data from areas through mobile devices.
  • Mobile devices uses ODK Collect application for collecting surveys.
  • Data gathered from surveys managed by ASIMS.

What is ODK Collect?

  • ODK Collect Â is an open source Android app that replaces paper forms used in survey-based data gathering.
  • We are using ODK Collect for collect data from fields and then receive it at the server (ASIMS).
  • ODK collect has ability to collect data offline
